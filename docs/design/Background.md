# reference projects

## Primary (significant community adoption or active development)

- [rust-ibverbs](https://github.com/jonhoo/rust-ibverbs) — ⭐203, ibverbs safe wrapper
- [async-rdma](https://github.com/datenlord/async-rdma) — ⭐429, async RDMA framework
- [rdma](https://github.com/Nugine/rdma) — ⭐39, low-level RDMA API
- [sideway](https://github.com/RDMA-Rust/sideway) — ⭐77, new ibverbs API wrapper (actively maintained)
- [rrddmma](https://github.com/IcicleF/rrddmma) — ⭐13, Mellanox-focused academic library

## FFI / Sys Crates

- [rdma-sys](https://github.com/datenlord/rdma-sys) — ⭐49, raw FFI bindings (used by async-rdma)
- [rdma-mummy-sys](https://github.com/RDMA-Rust/rdma-mummy-sys) — FFI bindings (used by sideway)

## Inactive / Historical

- [rdma-core](https://github.com/lemonrock/rdma-core) — ⭐18, experimental (last commit: 2018)
- [rdma-rs](https://github.com/phoenix-dataplane/rdma-rs) — ⭐4, ibverbs+rdmacm wrapper (incomplete, from Phoenix dataplane)

See [ExistingLibs.md](ExistingLibs.md) for detailed analysis.
